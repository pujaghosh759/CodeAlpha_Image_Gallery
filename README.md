#  Image Gallery Project

This project is a responsive **Image Gallery** built using **HTML, CSS, and JavaScript**.  
It includes a lightbox viewer, next/previous navigation, smooth transitions, hover effects, and category-based image filtering.

---

##  Features

### Responsive Grid Layout  
- Auto-fit images for all screen sizes  
- Mobile-friendly design  

###  Lightbox View  
- Click any image to view in fullscreen mode  
- Next / Previous navigation buttons  
- Dark overlay background  

### Hover Effects  
- Smooth scaling  
- Fade effect on hover  

###  Category Filters  
- Filter by Nature, City, Animals

###  Smooth Transitions  
- Clean animations for hover & filter

##  How to Run the Project

### **On Mobile (Acode / Any Editor):**
1. Create a folder  
2. Add all three files:  
   - `index.html`  
   - `style.css`  
   - `script.js`  
3. Open `index.html` in **Live Preview**  
4. The gallery will run perfectly

### **On Desktop:**
1. Download/clone the repo  
2. Open `index.html` in any browser  
3. No server required
